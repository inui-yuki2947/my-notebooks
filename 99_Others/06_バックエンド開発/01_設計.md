# Others/バックエンド開発/設計

## APIメソッド

| メソッド | 概要                                                         |
| -------- | ------------------------------------------------------------ |
| GET      | リソースを取得する。                                         |
| POST     | リソースを作成する。                                         |
| PUT      | リソースを上書きする。（既存リソースが存在しなければ作成する） |
| PATCH    | リソースを一部更新する。                                     |
| DELETE   | リソースを削除する。                                         |
