# Ruby/共通/制御構造

## 条件判断

### if, unless

- 書式（複数行）

  ```ruby
  if|unless 条件1 [then]
    処理1
  elsif 条件2 [then]
    処理2
  else
    処理3
  end
  ```

- 書式（1行）

  ```ruby
  処理 if|unless 条件
  ```

### case

- 書式

  ```ruby
  case 比較したいオブジェクト
  when 値1 [then]
    処理1
  ...
  else 
    処理n
  end
  ```

- `case`文は、`===`演算子（`==`より判定が緩い演算子）で比較を行う。

## 繰り返し

### while, until

- 書式

  ```ruby
  while|until 条件 [do]
    処理
  end
  ```

### loop

- 書式

  ```ruby
  loop [do]
    処理
  end
  ```

### times

- 書式

  ```ruby
  繰り返す回数.times do
    処理
  end
  ```

### for, each

- 書式（for）

  ```ruby
  for 変数 in オブジェクト [do]
    処理
  end
  ```

- 書式（each）

  ```ruby
  オブジェクト.each do |変数|
    処理
  end
  ```

- `開始..終了`で、範囲を表すオブジェクトを生成する。
