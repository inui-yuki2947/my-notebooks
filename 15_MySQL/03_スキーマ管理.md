# スキーマ管理

## SHOW DATABASES

### 記法

| 記法             | 概要                                       |
| ---------------- | ------------------------------------------ |
| `SHOW DATABASES` | データベース（スキーマ）の一覧を表示する。 |

## information_schema

### テーブル

| テーブル  | 概要                                 |
| --------- | ------------------------------------ |
| `TABLES`  | テーブル一覧の情報が格納されている。 |
| `COLUMNS` | カラム一覧の情報が格納されている。   |
