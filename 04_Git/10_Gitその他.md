## rev-parse

### コマンド

| コマンド                                | 概要                         |
| --------------------------------------- | ---------------------------- |
| `git rev-parse ブランチ名`              | ハッシュ値を表示する。       |
| `git rev-parse --abbrev-ref ブランチ名` | 正式なブランチ名を表示する。 |

## submodule

### コマンド

| コマンド               | 概要                                             |
| ---------------------- | ------------------------------------------------ |
| `git submodule`        | サブモジュール一覧を表示する。                   |
| `git sudmodule update` | サブモジュールを親で指定したコミットに更新する。 |
