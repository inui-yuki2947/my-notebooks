# その他

## nmap

### コマンド

| コマンド                    | 概要                                   |
| --------------------------- | -------------------------------------- |
| `nmap FQDNまたはIPアドレス` | 指定したホストのポートスキャンを行う。 |

## nethogs, iftop

### 概要

- `nethogs`コマンドで、どのプロセスの通信量が多いかを調べることができる。
- `iftop`コマンドで、どの通信相手との通信量が多いかを調べることができる。

## nc

### コマンド

| コマンド                     | 概要                                              |
| ---------------------------- | ------------------------------------------------- |
| `nc [ホスト名] [ポート番号]` | `cat`コマンドと同様の働きをネットワーク上で行う。 |

### オプション

| オプション          | 概要                           |
| ------------------- | ------------------------------ |
| `-l`                | 指定したポートをリッスンする。 |
| `-o 出力ファイル名` | 指定したファイルに出力する。   |
