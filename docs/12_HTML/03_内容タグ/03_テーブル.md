# HTML/内容タグ/テーブル

## table, tr, th, td

### タグ

| タグ                  | 概要                         |
| --------------------- | ---------------------------- |
| `<table>内容</table>` | テーブル外枠のタグ。         |
| `<tr>内容</tr>`       | `<table>`タグ内での1行。     |
| `<th>内容</th>`       | `<tr>`タグ内での見出しセル。 |
| `<td>内容</td>`       | `<tr>`タグ内でのデータセル。 |

## thead, tbody, tfoot

### タグ

| タグ                  | 概要                          |
| --------------------- | ----------------------------- |
| `<thead>内容</thead>` | `<table>`タグ内でのヘッダー。 |
| `<tbody>内容</tbody>` | `<table>`タグ内でのボディ。   |
| `<tfoot>内容</tfoot>` | `<table>`タグ内でのフッター。 |

## その他

### 例

```html
<table>
    <thead>
    <tr>
        <th>ヘッダーセル1</th>
        <th>ヘッダーセル2</th>
    </tr>
    </thead>

    <tbody>
    <tr>
        <td>セル1-1</td>
        <td>セル1-2</td>
    </tr>
    <tr>
        <td>セル2-1</td>
        <td>セル2-2</td>
    </tr>
    </tbody>

    <tfoot>
    <tr>
        <td>フッターセル1</td>
        <td>フッターセル2</td>
    </tr>
    </tfoot>
</table>
```
