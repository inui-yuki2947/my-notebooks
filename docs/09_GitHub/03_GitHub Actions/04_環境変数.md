## コンテキストと環境変数

| 種類         | 評価タイミング           | 使い時                           |
| ------------ | ------------------------ | -------------------------------- |
| コンテキスト | 仮想マシンに送られる前。 | `if`条件で使用したい場合など。   |
| 環境変数     | 仮想マシンでの実行時。   | コマンド内で使用したい場合など。 |
