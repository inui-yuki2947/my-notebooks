# GitHub/GitHub CLI/ヘルプ

## gh --version

### コマンド

| コマンド                                                     | 概要                                 |
| ------------------------------------------------------------ | ------------------------------------ |
| `gh --version`                                               | GitHub CLIのバージョンを表示する。   |

## gh help

### コマンド

| コマンド                                                     | 概要                                 |
| ------------------------------------------------------------ | ------------------------------------ |
| `gh help [コマンド [サブコマンド]]`<br />`gh [コマンド [サブコマンド]] --help` | 指定したコマンドのヘルプを表示する。 |
