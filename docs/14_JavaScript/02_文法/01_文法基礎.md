# JavaScript/文法/文法基礎

## 構文

- 各文末には、セミコロンを付けることが望ましい。（必須ではない）

## コメント

### 記法

| 記法              | 概要                           |
| ----------------- | ------------------------------ |
| `// コメント`     | 単一行コメント。               |
| `/* コメント */`  | 複数行コメント。               |
| `/** コメント */` | ドキュメンテーションコメント。 |
