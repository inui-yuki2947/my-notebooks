## Vuex

- ミューテーション
  - 「store.commit」がトリガーとなる。
  - ミューテーションは同期的でなければならない。
- アクション
  - 「store.dispatch」がトリガーとなる。
  - 任意の非同期処理を含むことができる。
- 型付け
  [Nuxt.js + TypeScript + Vuexをvuex-module-decoratorsでがっちりインテリセンスを効かせる - Qiita](https://qiita.com/azukiazusa/items/a50b1ffe05d9937a4db0)
