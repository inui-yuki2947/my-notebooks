## node

### コマンド

| コマンド                   | 概要                                 |
| -------------------------- | ------------------------------------ |
| `node`                     | Node.jsの対話的環境を起動する。      |
| `node JS形式ファイル`      | 指定したJS形式ファイルを実行する。   |
| `node -e JS形式スクリプト` | 指定したJS形式スクリプトを実行する。 |
