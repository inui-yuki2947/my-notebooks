## 共通

### 構文

- 各文末には、セミコロンを付けることが望ましい。（必須ではない）

### インデント

- インデントは、スペース2個を使うことが多い。

## コメント

### 記法

| 記法              | 概要                           |
| ----------------- | ------------------------------ |
| `// コメント`     | 単一行コメント。               |
| `/* コメント */`  | 複数行コメント。               |
| `/** コメント */` | ドキュメンテーションコメント。 |
