# AWS/共通/CodePipeline

## aws codepipeline

### コマンド

| コマンド                                          | 概要                                               |
| ------------------------------------------------- | -------------------------------------------------- |
| `aws codepipeline list-pipelines`                 | コードパイプラインの一覧を表示する。               |
| `aws codepipeline get-pipeline --name 名前`       | 指定したコードパイプラインの情報を表示する。       |
| `aws codepipeline get-pipeline-state --name 名前` | 指定したコードパイプラインのステータスを表示する。 |
