## 日時

### 概要

- 日時の情報は、ISO 8601拡張形式が推奨？

### ISO 8601

| 書式                        | 概要              |
| --------------------------- | ----------------- |
| `yyyyMMddTHHmmssZ`          | 基本形式（UTC）。 |
| `yyyyMMddTHHmmss+0900`      | 基本形式（JST）。 |
| `yyyy-MM-ddTHH:mm:ssZ`      | 拡張形式（UTC）。 |
| `yyyy-MM-ddTHH:mm:ss+09:00` | 拡張形式（JST）。 |

- [ISO 8601 - Wikipedia](https://ja.wikipedia.org/wiki/ISO_8601)
