# MySQL/関数と演算子/情報関数

## DATABASE()

### 書式

| 書式         | 概要                                           |
| ------------ | ---------------------------------------------- |
| `DATABASE()` | 現在のデフォルトデータベースの名前を取得する。 |

## USER(), CURRENT_USER()

### 書式

| 書式             | 概要                                                         |
| ---------------- | ------------------------------------------------------------ |
| `USER()`         | 接続時に指定したユーザー名とホスト名を返す。                 |
| `CURRENT_USER()` | 実際に認証されたユーザー名とホスト名を返す。（通常は`USER()`と同じ） |

## LAST_INSERT_ID()

### 書式

| 書式               | 概要                                                         |
| ------------------ | ------------------------------------------------------------ |
| `LAST_INSERT_ID()` | 最後に実行された INSERT ステートメントの結果として、<br />**AUTO_INCREMENTカラム**に正常に挿入された最初の自動生成値を返す。 |
