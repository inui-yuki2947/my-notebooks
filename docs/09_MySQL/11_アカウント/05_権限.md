# MySQL/アカウント/権限

## SHOW GRANTS

### 書式

| 書式                                                         | 概要                                 |
| ------------------------------------------------------------ | ------------------------------------ |
| `SHOW GRANTS`<br />`SHOW GRANTS FOR CURRENT_USER`<br />`SHOW GRANTS FOR CURRENT_USER()` | 現在のアカウントの権限を表示する。   |
| `SHOW GRANTS FOR アカウント名`                               | 指定したアカウントの権限を表示する。 |
