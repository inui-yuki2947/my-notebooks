# MySQL/プリペアドステートメント/共通

## PREPARE

### 書式

| 書式                                           | 概要                               |
| ---------------------------------------------- | ---------------------------------- |
| `PREPARE ステートメント名 FROM ステートメント` | 準備済みステートメントを定義する。 |

### ステートメント

- `?`と記述すると、後から変数を埋め込める。

## DEALLOCATE, DROP PREPARE

### 書式

| 書式                                          | 概要                                       |
| --------------------------------------------- | ------------------------------------------ |
| `<DEALLOCATE｜DROP> PREPARE ステートメント名` | 指定した準備済みステートメントを削除する。 |

## EXECUTE

### 書式

| 書式                                             | 概要                                                         |
| ------------------------------------------------ | ------------------------------------------------------------ |
| `EXECUTE ステートメント名 [USING @変数名1, ...]` | （指定した変数を埋め込み、）準備済みステートメントを実行する。 |
