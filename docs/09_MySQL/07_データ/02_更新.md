# MySQL/データ/更新

## UPDATE

### 書式

| 書式                                   | 概要                  |
|--------------------------------------|---------------------|
| `UPDATE テーブル名 SET <カラム名1 = 値1>, ...` | 指定したテーブルのレコードを更新する。 |

### 書式オプション

| 書式オプション   | 概要                                           |
| ---------------- | ---------------------------------------------- |
| `... WHERE 条件` | 指定した条件に当てはまるレコードのみ更新する。 |
