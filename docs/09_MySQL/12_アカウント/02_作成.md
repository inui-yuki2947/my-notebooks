# MySQL/アカウント/作成

## CREATE USER

### 書式

| 書式                               | 概要                   |
| ---------------------------------- | ---------------------- |
| `CREATE USER アカウント定義1, ...` | アカウントを作成する。 |

### アカウント定義

| アカウント定義                                     | 概要                                                         |
| -------------------------------------------------- | ------------------------------------------------------------ |
| `アカウント名`                                     | 暗黙的なプラグインを割り当てた、パスワードのないアカウント。 |
| `アカウント名 IDENTIFIED WITH プラグイン`          | 指定したプラグインを割り当てた、パスワードのないアカウント。 |
| `アカウント名 IDENTIFIED BY [PASSWORD] パスワード` | 暗黙的なプラグインを割り当てた、パスワードのあるアカウント。 |
