# Ruby/CUI/実行

## ruby

### コマンド

| コマンド                 | 概要                             |
| ------------------------ | -------------------------------- |
| `ruby`                   | 標準入力を実行する。             |
| `ruby ファイル名 [引数]` | 指定したrubyファイルを実行する。 |
