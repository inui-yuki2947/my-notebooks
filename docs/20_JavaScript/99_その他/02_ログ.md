# JavaScript/その他/ログ

## console

### 書式

| 書式                                | 概要                                                        |
| ----------------------------------- | ----------------------------------------------------------- |
| `console.log(オブジェクト1, ...)`   | 指定したオブジェクトを、コンソールに出力する。              |
| `console.info(オブジェクト1, ...)`  | 指定したオブジェクトを、infoレベルでコンソールに出力する。  |
| `console.warn(オブジェクト1, ...)`  | 指定したオブジェクトを、warnレベルでコンソールに出力する。  |
| `console.error(オブジェクト1, ...)` | 指定したオブジェクトを、errorレベルでコンソールに出力する。 |
