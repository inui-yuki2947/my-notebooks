# JavaScript/制御構造/条件分岐

## if

```js
if (条件) {
  処理1
} else if (条件2) {
  処理2
...
} else {
  処理3
}
```

- 条件を満たす場合に、処理を実行する。

## switch

```js
switch (対象) {
  case 値1:
    処理1
    break
  case 値2:
    処理2
    break
  ...
  default:
    処理3
    break
}
```

- `対象 === 値`である場合に、処理を実行する。

## 三項演算子

```js
条件 ? 値1 : 値2
```

- 条件を満たす場合は値1を、満たさない場合は値2を返す。
