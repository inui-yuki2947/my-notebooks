# JavaScript/文法/例外処理

## try catch

### 書式

| 書式  | 概要                                                         |
| ----- | ------------------------------------------------------------ |
| 書式1 | 処理1を実行し、<br />エラーが起きた場合は処理2を実行し、<br />最後に必ず処理3を実行する。 |

- 書式1

  ```js
  try {
    処理1
  } catch (変数) {
    処理2
  } finally {
    処理3
  }
  ```

## throw

### 書式

| 書式         | 概要               |
| ------------ | ------------------ |
| `throw 例外` | 例外をスローする。 |
