# JavaScript/文法/条件判断

## falsyな値

- 以下の値は`false`とみなされる。
  - 空文字
  - `0`
  - `NaN`
  - `null`
  - `undefined`

## 比較演算子

| 書式          | 概要                                                     |
| ------------- | -------------------------------------------------------- |
| `値1 == 値2`  | 値1と値2が等しいか。（データ型なども変換しつつ比較する） |
| `値1 === 値2` | 値1と値2が等しいか。（厳密に比較する）                   |

## 論理演算子

| 書式               | 概要                          |
| ------------------ | ----------------------------- |
| `! 条件`           | 条件を満たさないか。          |
| `条件1 && 条件2`   | 条件1,2をともに満たすか。     |
| `条件1 ｜｜ 条件2` | 条件1,2のどちらかを満たすか。 |
