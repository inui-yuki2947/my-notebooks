# JavaScript/npm,yarn/タスク管理

## タスク

### 概要

- タスク（npm-scripts）は、`package.json`ファイルの`scripts`に記述する。

### 書式

```json
{
  ...
  "scripts": {
    "タスク名1": "コマンド1",
    "タスク名2": "コマンド2",
    ...
  },
  ...
}
```

## npm run

### コマンド

| コマンド           | 概要                       |
| ------------------ | -------------------------- |
| `npm run`          | タスクの一覧を表示する。   |
| `npm run タスク名` | 指定したタスクを実行する。 |

### その他

- `preタスク名`というタスクが存在すれば、タスク実行前にそのタスクが実行される。
- `postタスク名`というタスクが存在すれば、タスク実行後にそのタスクが実行される。
