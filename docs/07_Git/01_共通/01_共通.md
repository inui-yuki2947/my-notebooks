## インストール

### Windows

- [Git - Downloading Package](https://git-scm.com/download/win)

### Mac

```bash
brew install git
```

### Ubuntu

```bash
sudo add-apt-repository ppa:git-core/ppa
sudo apt update
sudo apt install git
```
