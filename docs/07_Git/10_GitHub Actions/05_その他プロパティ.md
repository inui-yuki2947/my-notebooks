# Git/GitHub Actions/その他プロパティ

## name

### プロパティ

| プロパティ | 型       | 概要                                                         | デフォルト値                                             |
| ---------- | -------- | ------------------------------------------------------------ | -------------------------------------------------------- |
| `name`     | 文字列型 | GitHubリポジトリの「Actions」タブに表示される、<br />ワークフローの名前。 | リポジトリルートに対する、<br />当該ファイルの相対パス。 |

## env

### プロパティ

| プロパティ | 型                                  | 概要                                                 |
| ---------- | ----------------------------------- | ---------------------------------------------------- |
| `env`      | オブジェクト型<br />（Key/Value型） | 環境変数として設定する値。（ワークフロー内スコープ） |
