## 共通

### 概要

- [GitHub Actions Documentation - GitHub Docs](https://docs.github.com/en/actions)

## 設定ファイル

### 事前準備

```bash
mkdir -p .github/workflows
```

### ファイル

| ファイル                  | 概要                                       |
| ------------------------- | ------------------------------------------ |
| `.github/workflows/*.yml` | 各ワークフローについて記述されたファイル。 |

### 書式

```yml
name: ワークフロー名（任意）
on: [ワークフローのトリガー]
jobs:
  ジョブ名:
    runs-on: 実行環境
    steps:
      - name: 名前
        uses: 使用するアクション
        run: 実行するコマンド
         ...
```
