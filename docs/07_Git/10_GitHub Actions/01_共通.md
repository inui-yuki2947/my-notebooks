# Git/GitHub Actions/共通

## 共通

### 事前準備

```bash
mkdir -p .github/workflows
```

### ファイル

| ファイル                  | 概要                                       |
| ------------------------- | ------------------------------------------ |
| `.github/workflows/*.yml` | 各ワークフローについて記述されたファイル。 |

### 書式

```yml
name: ワークフロー名（任意）
on: [ワークフローのトリガー]
jobs:
  ジョブ名:
    runs-on: 実行環境
    steps:
      - name: 名前
        uses: 使用するアクション
        run: 実行するコマンド
         ...
```

### ワークフロー・ジョブ・ステップ

| 名前         | 概要                                 |
| ------------ | ------------------------------------ |
| ワークフロー | 実行の最大単位。                     |
| ジョブ       | ワークフロー内で並列実行される処理。 |
| ステップ     | ジョブ内で直列実行される処理。       |
