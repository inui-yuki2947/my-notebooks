# Go/CUI/gore

## gore

### コマンド

| コマンド | 概要             |
| -------- | ---------------- |
| `gore`   | goreを起動する。 |

### 操作

| 操作                 | 概要                         |
| -------------------- | ---------------------------- |
| `:import パッケージ` | パッケージをインポートする。 |
| `:quit`              | セッションを終了する。       |
