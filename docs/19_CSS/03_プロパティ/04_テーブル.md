# CSS/プロパティ/テーブル

## 共通

### プロパティ

| プロパティ     | 概要                        |
| -------------- | --------------------------- |
| `table-layout` | `<table>`タグのレイアウト。 |

### 値（table-layout）

| 値      | 概要                       |
| ------- | -------------------------- |
| `auto`  | 自動レイアウト。           |
| `fixed` | 列幅を固定したレイアウト。 |
