# TypeScript/CUI/tscofig

## ファイル

| ファイル        | 概要                                   |
| --------------- | -------------------------------------- |
| `tsconfig.json` | TypeScriptプロジェクトの設定ファイル。 |

## 例

```json
{
  "compilerOptions": {
    "target": "es6",
    "module": "commonjs",
    "strict": true
  },
  "include": ["src/**/*"]
}
```
