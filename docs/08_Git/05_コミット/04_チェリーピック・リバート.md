# Git/コミット/チェリーピック・リバート

## git cherry-pick

### コマンド

| コマンド                        | 概要                                           |
| ------------------------------- | ---------------------------------------------- |
| `git cherry-pick コミット1 ...` | 指定したコミットを（全て）チェリーピックする。 |

## git revert

### コマンド

| コマンド              | 概要                 |
| --------------------- | -------------------- |
| `git revert コミット` | コミットを打ち消す。 |

### オプション

| オプション                       | 概要                                   |
| -------------------------------- | -------------------------------------- |
| `--mainline, -m <parent-number>` | マージコミットを打ち消すことができる。 |

- `<parent-number>`は、`1`がマージされた側のブランチ、`2`がマージする側のブランチ。（なので基本的には`1`を指定する）
