# Go/その他/ファイル

## os

### メソッド

| メソッド     | 概要                             |
| ------------ | -------------------------------- |
| `os.Getwd()` | カレントディレクトリを取得する。 |

## filepath

### メソッド

| メソッド              | 概要                                     |
| --------------------- | ---------------------------------------- |
| `filepath.Ext(パス)`  | 指定したパスの拡張子を取得する。         |
| `filepath.Base(パス)` | 指定したパスのファイル名を取得する。     |
| `filepath.Dir(パス)`  | 指定したパスのディレクトリ名を取得する。 |

## runtime

### メソッド

| メソッド                     | 概要                                                         |
| ---------------------------- | ------------------------------------------------------------ |
| `runtime.Caller(スキップ数)` | 当メソッドが呼び出されたファイル名・行番号などを取得する。<br />スキップで指定した数だけ、呼び出し元を辿る。 |
