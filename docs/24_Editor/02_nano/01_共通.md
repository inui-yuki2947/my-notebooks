## nano

### コマンド

| コマンド             | 概要                                 |
| -------------------- | ------------------------------------ |
| `nano`               | 新しいファイルを、nanoで編集する。   |
| `nano ファイル名`    | 指定したファイルを、nanoで編集する。 |
| `nano -V, --version` | nanoのバージョンを表示する。         |
| `nano -h, --help`    | nanoのヘルプを表示する。             |
