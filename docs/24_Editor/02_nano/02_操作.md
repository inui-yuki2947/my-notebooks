# Editor/nano/操作

## 操作

| 分類       | 操作                 | 概要                                                         |
| ---------- | -------------------- | ------------------------------------------------------------ |
| 共通       | `F1`<br />`Ctrl + G` | ヘルプを表示する。                                           |
| 起動・終了 | `F2`<br />`Ctrl + X` | nanoを終了する。                                             |
| 起動・終了 | `F3`<br />`Ctrl + O` | 編集内容を保存する。<br />（ファイル名の入力が求められるので、入力後Enterを押下する） |
| 編集       | `Ctrl + K`           | 現在の行を切り取る。                                         |
| 編集       | `Ctrl + U`           | コピーした内容を貼り付ける。                                 |
