# Perl/その他/引数処理

## getopts

### 書式

| 書式                                                    | 概要                                 |
| ------------------------------------------------------- | ------------------------------------ |
| `getopts(受け取るオプション文字列, 代入先ハッシュ変数)` | オプションを解析し、変数に代入する。 |

- `use Getopt::Std`が必要。
- [オプションの取得 - Perl表技集](https://mas3lab.net/perl/waza/opt_std.html)
