# Perl/その他/その他

## exit

### 記法

| 記法                   | 概要                                           |
| ---------------------- | ---------------------------------------------- |
| `exit(終了ステータス)` | 指定した終了ステータスでスクリプトを終了する。 |

## defined

### 記法

| 記法          | 概要                                     |
| ------------- | ---------------------------------------- |
| `defined(値)` | 値が未定義であるかどうかを真偽値で返す。 |

## eval

### 記法

| 記法       | 概要                               |
| ---------- | ---------------------------------- |
| `eval(値)` | 指定した値をコードとして評価する。 |
