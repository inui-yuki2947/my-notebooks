# Perl/文法/共通

## 概要

| 項目       | 内容                                         |
| ---------- | -------------------------------------------- |
| 拡張子     | `.pl`                                        |
| 文の区切り | `;`（必須）                                  |
| インデント | スペース2個または4個                         |
| シバン     | `#!/usr/bin/perl`<br />`#!/usr/bin/env perl` |

## コメント

```perl
# コメント
```

## 文法のチェック

```perl
use strict;
use warnings;
```

## 文字列

- 文字列は、シングルクォーテーションまたはダブルクォーテーションで囲う。
