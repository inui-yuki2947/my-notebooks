# Perl/文法/変数

## 定義・代入・参照

### 書式（定義）

| 書式         | 概要                                           |
| ------------ | ---------------------------------------------- |
| `変数名;`    | 変数を定義する。<br />（strictモードでは不可） |
| `my 変数名;` | 変数を定義する。                               |

### 書式（代入）

| 書式              | 概要                                                         |
| ----------------- | ------------------------------------------------------------ |
| `変数名 = 値;`    | 変数に代入する。<br />（strictモードでは、未定義変数への代入は不可） |
| `my 変数名 = 値;` | 変数を定義して代入する。                                     |

### 書式（参照）

| 書式     | 概要                 |
| -------- | -------------------- |
| `変数名` | 変数の値を参照する。 |

### 変数名

| 書式        | 概要                                |
| ----------- | ----------------------------------- |
| `$var_name` | スカラ変数。                        |
| `@var_name` | 配列変数。（インデックスは0始まり） |
| `%var_name` | ハッシュ変数。                      |

## 定義済み変数

### 書式

| 書式 | 概要                     |
| ---- | ------------------------ |
| `$_` | 最後に読み込んだレコード |
| `$.` | レコード番号             |
| `$/` | レコードの区切り文字     |
