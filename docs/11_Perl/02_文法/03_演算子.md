# Perl/文法/演算子

## 比較演算子

| 書式                 | 概要                     |
| -------------------- | ------------------------ |
| `数値1 == 数値2`     | 数値が等しいかどうか。   |
| `文字列1 eq 文字列2` | 文字列が等しいかどうか。 |

## 三項演算子

| 書式               | 概要                                                 |
| ------------------ | ---------------------------------------------------- |
| `条件 ? 値1 : 値2` | 条件を満たす場合は値1を、満たさない場合は値2を返す。 |
