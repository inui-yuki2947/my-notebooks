# Perl/制御構造/条件分岐

## if

```perl
if (条件) {
    処理
} elsif (条件) {
    処理
} else {
    処理
}
```

```perl
処理 if 条件
```

- 条件を満たす場合に、処理を実行する。

## unless

```perl
unless (条件) {
    処理
}
```

```perl
処理 unless 条件
```

- 条件を満たさない場合に、処理を実行する。

## 三項演算子

```perl
条件 ? 値1 : 値2
```

- 条件を満たす場合は値1を、満たさない場合は値2を返す。
