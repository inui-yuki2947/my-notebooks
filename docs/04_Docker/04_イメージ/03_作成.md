# Docker/イメージ/作成

## docker image build

### コマンド

| コマンド                          | 概要                                                         |
| --------------------------------- | ------------------------------------------------------------ |
| `docker image build ディレクトリ` | 指定したディレクトリにある**Dockerfile**をもとに、イメージを作成する。 |

### オプション

| オプション              | 概要                                |
| ----------------------- | ----------------------------------- |
| `-t イメージ`           | イメージ名とタグ名を指定する。      |
| `--build-arg 変数名=値` | ARG句で定義した変数の値を指定する。 |
| `--progress=plain`      | ログを出力する。                    |
