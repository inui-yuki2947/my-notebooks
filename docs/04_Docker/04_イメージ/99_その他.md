# Docker/イメージ/その他

## docker image tag

### コマンド

| コマンド                                   | 概要                       |
| ------------------------------------------ | -------------------------- |
| `docker image tag イメージ 新しいイメージ` | イメージにタグを付与する。 |

## docker image pull, push

### コマンド

| コマンド                     | 概要                                 |
| ---------------------------- | ------------------------------------ |
| `docker image pull イメージ` | レジストリからイメージを取得する。   |
| `docker image push イメージ` | レジストリにイメージをプッシュする。 |
