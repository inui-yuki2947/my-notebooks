# Docker/ボリューム/共通

## docker volume

### 概要

- ボリュームは、ホストのファイルシステムの一部としてデータが保存される。
- 保存先は`/var/lib/docker/volumes/`であるが、そのパスでなく任意に付けた名前を使用する。

### コマンド

| コマンド                            | 概要                   |
| ----------------------------------- | ---------------------- |
| `docker volume create ボリューム名` | ボリュームを作成する。 |
