## docker -v, version

### コマンド

| コマンド               | 概要                                 |
| ---------------------- | ------------------------------------ |
| `docker -v, --version` | Dockerのバージョンを表示する。       |
| `docker version`       | Dockerのバージョンを詳しく表示する。 |

## docker --help

### コマンド

| コマンド                   | 概要                               |
| -------------------------- | ---------------------------------- |
| `docker [コマンド] --help` | Dockerコマンドのヘルプを表示する。 |
