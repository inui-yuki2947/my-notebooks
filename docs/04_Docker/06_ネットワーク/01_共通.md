# Docker/ネットワーク/共通

## 共通

### 概要

- 複数のコンテナを一つのネットワークに繋げると、それらは通信できるようになる。
- Dockerインストール時に、自動的に3つのネットワークが作成されている。

## docker network ls

### コマンド

| コマンド            | 概要                           |
| ------------------- | ------------------------------ |
| `docker network ls` | ネットワークの一覧を表示する。 |

## docker network create

### コマンド

| コマンド                               | 概要                           |
| -------------------------------------- | ------------------------------ |
| `docker network create ネットワーク名` | 新しいネットワークを作成する。 |
