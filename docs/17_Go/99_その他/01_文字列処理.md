# Go/その他/文字列処理

## 正規表現

### 記法

| 記法                                 | 戻り値    | 概要                                   |
| ------------------------------------ | --------- | -------------------------------------- |
| `regexp.MustCompile(str string)`     | `*Regexp` | 正規表現オブジェクトを生成する。       |
| `(re *Regexp) MatchString(s string)` | `bool`    | 文字列が正規表現にマッチするかを返す。 |
