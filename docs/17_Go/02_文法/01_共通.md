# Go/文法/共通

## 共通

## 概要

| 項目       | 内容        |
| ---------- | ----------- |
| 拡張子     | `.go`       |
| 文の区切り | `;`（任意） |
| インデント | タブ        |

### Go言語の特徴

- ネイティブコードにコンパイルされる。
- マルチプラットフォームで動作する。（OSの標準ライブラリにすら依存しない）
- オブジェクト指向を有さず、関数と構造体で記述する。
- 使用されていない`import`文や変数があると、厳密にコンパイルエラーを出す。

## コメント

### 単一行

```go
// コメント
```

### 複数行

```go
/*
コメント1
コメント2
...
*/
```
