# Git/その他/その他

## git reflog

### コマンド

| コマンド     | 概要                      |
| ------------ | ------------------------- |
| `git reflog` | Gitの操作履歴を表示する。 |

## git rev-parse

### コマンド

| コマンド                 | 概要                                     |
| ------------------------ | ---------------------------------------- |
| `git rev-parse コミット` | 指定したコミットのハッシュ値を表示する。 |

### オプション

| オプション       | 概要                           |
| ---------------- | ------------------------------ |
| `--abbrev-ref`   | 正式なブランチ名を表示する。   |
| `--short [長さ]` | 短縮版のハッシュ値を表示する。 |
