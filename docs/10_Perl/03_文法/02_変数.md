# Perl/文法/変数

## 共通

### 記法

| 記法         | 概要                     |
| ------------ | ------------------------ |
| `my $変数名` | スカラ変数を定義する。   |
| `my @変数名` | 配列変数を定義する。     |
| `my %変数名` | ハッシュ変数を定義する。 |

### 定義済み変数

| 定義済み変数 | 概要                     |
| ------------ | ------------------------ |
| `$_`         | 最後に読み込んだレコード |
| `$.`         | レコード番号             |
| `$/`         | レコードの区切り文字     |
