# AWS/コンテナ/ECS

## 共通

### 概要

- ECS (Elastic Container Service)は、AWSのフルマネージドコンテナオーケストレーションサービス。

## aws ecs run-task

### コマンド

| コマンド                                        | 概要                                                 |
| ----------------------------------------------- | ---------------------------------------------------- |
| `aws ecs run-task --task-definition タスク定義` | 指定したタスク定義を用いて、新しいタスクを開始する。 |

## aws ecs describe-tasks

### コマンド

| コマンド                                | 概要                             |
| --------------------------------------- | -------------------------------- |
| `aws ecs describe-tasks --tasks タスク` | 指定したタスクの詳細を表示する。 |
