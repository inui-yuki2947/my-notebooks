# Bash/文字列処理/その他

## wc

### コマンド

| コマンド           | 概要                                                   |
| ------------------ | ------------------------------------------------------ |
| `wc`               | 標準入力の、行数・単語数・バイト数を表示する。         |
| `wc ファイル名...` | 指定したファイルの、行数・単語数・バイト数を表示する。 |

### オプション

| オプション    | 概要                   |
| ------------- | ---------------------- |
| `-l, --lines` | 行数のみ表示する。     |
| `-w, --words` | 単語数のみ表示する。   |
| `-c, --bytes` | バイト数のみ表示する。 |
