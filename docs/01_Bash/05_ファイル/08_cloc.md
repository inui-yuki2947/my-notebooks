# Bash/ファイル/cloc

## 共通

### 概要

- 「cloc」は、ファイルの行数を測定するツール。

### 参考資料

- [AlDanial/cloc: cloc counts blank lines, comment lines, and physical lines of source code in many programming languages.](https://github.com/AlDanial/cloc)

## インストール

### Mac

```bash
brew install cloc
```

## cloc

### コマンド

| コマンド    | 概要                           |
| ----------- | ------------------------------ |
| `cloc パス` | 指定したパスの行数を測定する。 |
