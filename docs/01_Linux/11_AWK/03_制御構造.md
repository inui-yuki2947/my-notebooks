# Linux/AWK/制御構造

## if

### 記法

| 記法                           | 概要                                 |
| ------------------------------ | ------------------------------------ |
| `if (条件) 処理1 [else 処理2]` | 条件を満たす場合に、処理を実行する。 |

## while

### 記法

| 記法                | 概要                                         |
| ------------------- | -------------------------------------------- |
| `while (条件) 処理` | 条件を満たす場合に、繰り返し処理を実行する。 |

## for

### 記法

| 記法                          | 概要                                         |
| ----------------------------- | -------------------------------------------- |
| `for (変数名 in 配列名) 処理` | 配列の要素について、繰り返し処理を実行する。 |
