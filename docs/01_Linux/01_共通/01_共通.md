## bash

### コマンド

| コマンド                     | 概要                           |
| ---------------------------- | ------------------------------ |
| `bash`                       | 標準入力のコマンドを実行する。 |
| `bash -c コマンド [引数...]` | 指定したコマンドを実行する。   |
| `bash ファイル名 [引数...]`  | 指定したファイルを実行する。   |

- コマンドは、子プロセスで実行される。
