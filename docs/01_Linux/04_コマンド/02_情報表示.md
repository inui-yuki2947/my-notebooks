# Linux/コマンド/情報表示

## type, which, whereis

### コマンド

| コマンド           | 概要                                                         |
| ------------------ | ------------------------------------------------------------ |
| `type コマンド`    | **実行されるコマンド**を表示する。                           |
| `which コマンド`   | コマンドが**PATHのどこにあるか**を表示する。                 |
| `whereis コマンド` | コマンドのバイナリファイル・ソースコード・マニュアルファイルが置かれている場所を表示する。 |
