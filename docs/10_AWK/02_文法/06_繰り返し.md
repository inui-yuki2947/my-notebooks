# AWK/文法/繰り返し

## while

```awk
while (条件) {
  処理
}
```

- 条件を満たす場合に、繰り返し処理を実行する。（先に条件評価）

## do while

```awk
do {
  処理
} while (条件)
```

- 条件を満たす場合に、繰り返し処理を実行する。（後に条件評価）

## for

```awk
for (初期化式; 条件; 増減式) {
  処理
}
```

- 初期化式を実行した後、条件を満たす限り、処理及び増減式を実行する。

## for in

```awk
for (変数名 in 配列名) {
  処理
}
```

- 配列の要素について、繰り返し処理を実行する。

## break, continue, next

| 書式       | 概要                       |
| ---------- | -------------------------- |
| `break`    | 繰り返しから抜ける。       |
| `continue` | 次の繰り返しに進む。       |
| `next`     | 次のレコードの処理に移る。 |
