# AWK/制御構造/条件分岐

## if

### 書式

| 書式                                                         | 概要                                 |
| ------------------------------------------------------------ | ------------------------------------ |
| `if (条件1) {処理1} [else if (条件2) {処理2}] ... [else {処理n}]` | 条件を満たす場合に、処理を実行する。 |

## その他

- 処理が1文である場合は、`{}`は省略可能。
