# 開発/IntelliJ/CLI

## idea

### 概要

- 「Toolbox App」を使うと、`idea`コマンド（`open -na "IntelliJ IDEA.app"`のエイリアス）が使用できるようになる。

### コマンド

| コマンド             | 概要                                          |
| -------------------- | --------------------------------------------- |
| `idea`               | IntelliJ IDEAを開く。                         |
| `idea ファイル名...` | IntelliJ IDEAで、指定したファイルを全て開く。 |
