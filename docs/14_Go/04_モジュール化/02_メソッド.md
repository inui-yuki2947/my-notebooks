# Go/モジュール化/メソッド

## メソッド

### 概要

- メソッドは、型に特化した関数のことである。

### 定義

- 書式

  ```go
  func (レシーバー レシーバーの型) 関数名(引数) 戻り値の型 {
    処理
  }
  ```

- 一般にレシーバーはポインタ型で定義する。

### 呼び出し

- メソッドを呼び出す場合

  ```go
  レシーバー.メソッド(引数)
  ```

- レシーバーは値型でもポインタ型でも、メソッドは区別せず呼び出せる。

### 関数の取り出し

- 下記書式で、メソッドを（レシーバーを第1引数とする）関数として取り出せる。

  ```go
  レシーバーの型.メソッド
  ```
