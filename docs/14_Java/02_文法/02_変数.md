# Java/文法/変数

## 定義・代入

### 書式

| 書式                               | 概要                     |
| ---------------------------------- | ------------------------ |
| `型 変数名`<br />`型 変数名1, ...` | 変数を定義する。         |
| `型 変数名 = 値`                   | 変数を定義して代入する。 |

- 先頭に`final`を付与すると、定数（再代入不可能な変数）を定義する。

## 参照

### 書式

| 書式     | 概要                 |
| -------- | -------------------- |
| `変数名` | 変数の値を参照する。 |
