# Python/文法/共通

## 概要

| 項目       | 内容                                                         |
| ---------- | ------------------------------------------------------------ |
| 拡張子     | `.py`                                                        |
| 文の区切り | `;`（任意）<br />（括弧などの中で、かつ単語ごとの区切りであれば、自由に改行が可能） |
| インデント | スペース4個<br />**（無条件のインデントは認められていない）** |
| シバン     | `#!/usr/bin/env python3`                                     |

## コメント

```python
# コメント
```

```python
"""
コメント
...
"""
```
