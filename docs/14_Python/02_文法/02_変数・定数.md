# Python/文法/変数・定数

## 定義・代入

| 書式          | 例                  | 概要                                         |
| ------------- | ------------------- | -------------------------------------------- |
| `変数名 = 値` | `var_name = 'hoge'` | 指定した変数に値を代入する。（事前定義不要） |
| `del 変数名`  | `del var_name`      | 変数を破棄する。                             |
