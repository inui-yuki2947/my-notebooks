# Go/コマンド/共通

## go version

### コマンド

| コマンド     | 概要                       |
| ------------ | -------------------------- |
| `go version` | goのバージョンを表示する。 |

## go help

### コマンド

| コマンド                 | 概要                       |
| ------------------------ | -------------------------- |
| `go help [サブコマンド]` | goのマニュアルを表示する。 |
