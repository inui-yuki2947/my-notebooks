# Ruby/制御構造/条件分岐

## 条件判断

### if, unless

- 書式（複数行）

  ```ruby
  if|unless 条件1 [then]
    処理1
  elsif 条件2 [then]
    処理2
  else
    処理3
  end
  ```

- 書式（1行）

  ```ruby
  処理 if|unless 条件
  ```

### case

- 書式

  ```ruby
  case 比較したいオブジェクト
  when 値1 [then]
    処理1
  ...
  else 
    処理n
  end
  ```

- `case`文は、`===`演算子（`==`より判定が緩い演算子）で比較を行う。
