## ruby -v

### コマンド

| コマンド             | 概要                         |
| -------------------- | ---------------------------- |
| `ruby -v, --version` | rubyのバージョンを表示する。 |

## ruby --help

### コマンド

| コマンド      | 概要                     |
| ------------- | ------------------------ |
| `ruby --help` | rubyのヘルプを表示する。 |

## ruby

### コマンド

| コマンド                 | 概要                             |
| ------------------------ | -------------------------------- |
| `ruby`                   | 標準入力を実行する。             |
| `ruby ファイル名 [引数]` | 指定したrubyファイルを実行する。 |

## irb

### コマンド

| コマンド | 概要                     |
| -------- | ------------------------ |
| `irb`    | 対話的にrubyを実行する。 |
