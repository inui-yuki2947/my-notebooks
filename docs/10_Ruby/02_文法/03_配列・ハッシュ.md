# Ruby/文法/配列・ハッシュ

## 配列

### 取得

- `配列.size`で配列の大きさを取得できる。

### 繰り返し

- 書式

  ```ruby
  配列.each do |変数|
    処理
  end
  ```

## ハッシュ

### 生成

- 通常の書式

  ```ruby
  {キー:> 値, ...}
  ```

- ハッシュを用いる場合の書式

  ```ruby
  {シンボル名: 値}
  ```

### 取得

- 書式

  ```ruby
  ハッシュ名[キー]
  ```

### 繰り返し

- 書式

  ```ruby
  ハッシュ.each do |キーの変数, 値の変数|
    処理
  end
  ```
