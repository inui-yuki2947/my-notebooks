# Ruby/文法/文法基礎

## 共通

### 文

- 各文の区切りには、セミコロンを用いる。
- 各文の区切りは必須ではない。

### インデント

- インデントは、スペース2個を使うことが多い。

## コメント

### 記法

| 記法         | 概要               |
| ------------ | ------------------ |
| `# コメント` | 単一行のコメント。 |
| 記法1        | 複数行のコメント。 |

- 記法1

  ```ruby
  =begin
  コメント
  ...
  =end
  ```
