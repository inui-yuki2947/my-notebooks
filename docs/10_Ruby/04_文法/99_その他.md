# Ruby/文法/その他

## ブロック

### 記法

| 記法                                | 概要                                               |
| ----------------------------------- | -------------------------------------------------- |
| `do ｜変数1, 変数2, ...｜ 処理 end` | ブロックを定義する。（複数行の場合はこちらが推奨） |
| `{｜変数1, 変数2, ...｜ 処理}`      | ブロックを定義する。（単一行の場合はこちらが推奨） |

### その他

- `&:method_name`で、ブロックの代わりにメソッド名を渡すことができる。
