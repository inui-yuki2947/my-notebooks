# Ruby/制御構造/繰り返し

## 繰り返し

### while, until

- 書式

  ```ruby
  while|until 条件 [do]
    処理
  end
  ```

### loop

- 書式

  ```ruby
  loop [do]
    処理
  end
  ```

### times

- 書式

  ```ruby
  繰り返す回数.times do
    処理
  end
  ```

### for, each

- 書式（for）

  ```ruby
  for 変数 in オブジェクト [do]
    処理
  end
  ```

- 書式（each）

  ```ruby
  オブジェクト.each do |変数|
    処理
  end
  ```

- `開始..終了`で、範囲を表すオブジェクトを生成する。
