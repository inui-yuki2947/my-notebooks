# Editor/nano/操作

## 共通

### 操作

| 操作                 | 概要               |
| -------------------- | ------------------ |
| `F1`<br />`Ctrl + G` | ヘルプを表示する。 |

## 起動・終了系

### 操作

| 操作                 | 概要                                                         |
| -------------------- | ------------------------------------------------------------ |
| `F2`<br />`Ctrl + X` | nanoを終了する。                                             |
| `F3`<br />`Ctrl + O` | 編集内容を保存する。<br />（ファイル名の入力が求められるので、入力後Enterを押下する） |

## 編集系

### 操作

| 操作       | 概要                         |
| ---------- | ---------------------------- |
| `Ctrl + K` | 現在の行を切り取る。         |
| `Ctrl + U` | コピーした内容を貼り付ける。 |
