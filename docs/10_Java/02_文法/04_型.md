# Java/文法/型

## プリミティブ型

### 真偽値型

| 型        | 概要     | 例              |
| :-------- | :------- | --------------- |
| `boolean` | 真偽値型 | `true`, `false` |

### 整数型

| 型      | 概要               | 例    |
| :------ | :----------------- | ----- |
| `byte`  | 整数型（8ビット）  | `123` |
| `short` | 整数型（16ビット） | `123` |
| `int`   | 整数型（32ビット） | `123` |
| `long`  | 整数型（64ビット） | `123` |

### 浮動小数点数型

| 型       | 概要                       | 例              |
| :------- | :------------------------- | --------------- |
| `float`  | 浮動小数点数型（32ビット） | `12.3F`         |
| `double` | 浮動小数点数型（64ビット） | `12.3`, `12.3D` |

### 文字列型

| 型     | 概要                            | 例    |
| :----- | :------------------------------ | ----- |
| `char` | 文字列型（Unicode文字16ビット） | `'A'` |

## 参照型

| 型       | 概要                   | 例      |
| :------- | :--------------------- | ------- |
| `Object` | 全ての参照型の先祖の型 |         |
| `String` | 文字列型               | `"ABC"` |
| `型[]`   | 配列型                 |         |
