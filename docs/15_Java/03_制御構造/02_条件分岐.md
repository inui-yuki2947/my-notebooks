# Java/制御構造/条件分岐

## if

```java
if (条件1) {
  処理1
} else if (条件2) {
  処理2
} ... else {
  処理n
}
```

- 条件を満たす場合に、処理を実行する。

## 三項演算子

```java
条件 ? 値1 : 値2
```

- 条件を満たす場合は値1を、満たさない場合は値2を返す。

## switch

```java
switch (表現) {
  case 値1:
    処理1
    break;
  ...
  default:
    処理n
}
```

- 表現が値と合致する場合に、処理を実行する。
  （`break`を記述しない場合は、後続のブロックが引き続き実行される）
