# Java/CUI/java

## java

### コマンド

| コマンド                 | 概要                             |
| ------------------------ | -------------------------------- |
| `java --version`         | Javaのバージョンを表示する。     |
| `java --help`            | Javaのヘルプを表示する。         |
| `java ファイル名 [引数]` | 指定したJavaファイルを実行する。 |
