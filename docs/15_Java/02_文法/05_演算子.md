# Java/文法/演算子

## 比較演算子

| 書式              | 概要                                           |
| ----------------- | ---------------------------------------------- |
| `値1 == 値2`      | 値1と値2が、同じインスタンスであるか。（等値） |
| `値1.equals(値2)` | 値1と値2が、意味的に同じ値であるか。（等価）   |

## 三項演算子

```java
条件 ? 値1 : 値2
```

- 条件を満たす場合は値1を、満たさない場合は値2を返す。
