# Java/文法/共通

## 共通

| 項目       | 内容                  |
| ---------- | --------------------- |
| 拡張子     | `.java`               |
| 文の区切り | `;`（必須）           |
| インデント | スペース4個またはタブ |

## コメント

### 単一行

```java
// コメント
```

### 複数行

```java
/* コメント1
コメント2
コメント3 */
```
