# Git/共通/CUI

## git version

### コマンド

| コマンド                           | 概要                        |
| ---------------------------------- | --------------------------- |
| `git version`<br />`git --version` | Gitのバージョンを表示する。 |

## git help

### コマンド

| コマンド                                               | 概要                                  |
| ------------------------------------------------------ | ------------------------------------- |
| `git help`<br />`git --help`                           | Gitのヘルプを表示する。               |
| `git help サブコマンド`<br />`git --help サブコマンド` | Gitのサブコマンドのヘルプを表示する。 |
