# Git/ブランチ・タグ/タグ

## git tag

### コマンド

| コマンド  | 概要                 |
| --------- | -------------------- |
| `git tag` | タグ一覧を表示する。 |

## git tag タグ名

### コマンド

| コマンド                                | 概要             |
| --------------------------------------- | ---------------- |
| `git tag タグ名 [タグを付けるコミット]` | タグを作成する。 |

### オプション

| オプション                 | 概要                           |
| -------------------------- | ------------------------------ |
| `-a, --annotate`           | メッセージ付きタグを作成する。 |
| `-m, --message メッセージ` | メッセージを付与する。         |
