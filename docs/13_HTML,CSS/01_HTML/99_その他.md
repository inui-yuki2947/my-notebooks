# HTML,CSS/HTML/その他

## h1, p

### タグ

| タグ                                          | 概要               |
| --------------------------------------------- | ------------------ |
| `<h1>内容</h1>`<br />...<br />`<h6>内容</h6>` | 見出し1〜6のタグ。 |
| `<p>内容</p>`                                 | 段落を表すタグ。   |

## input

### タグ

| タグ      | 概要                 |
| --------- | -------------------- |
| `<input>` | 入力フォームのタグ。 |

### タグ

| タグ                                                     | 概要                       |
