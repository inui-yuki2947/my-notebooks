# Ruby/文法/例外処理

## begin rescue ensure

```ruby
begin 
  処理1
[rescue [エラー型] [=> e]
  処理2-1
rescue [エラー型] [=> e]
  処理2-2
 ..]
[ensure
  処理3]
end
```

- 最初に処理1を実行し、例外が起きた場合は処理2を実行し、最後に必ず処理3を実行する。

## raise, retry

### 書式

| 書式    | 概要                                              |
| ------- | ------------------------------------------------- |
| `raise` | 例外を投げる。                                    |
| `retry` | 処理を`rescue`ブロックから`begin`ブロックに移す。 |
