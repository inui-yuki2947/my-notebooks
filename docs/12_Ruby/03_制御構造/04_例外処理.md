# Ruby/制御構造/例外処理

## begin~rescue~ensure

### 書式

| 書式  | 概要                                                         |
| ----- | ------------------------------------------------------------ |
| 書式1 | 最初に処理1を実行し、<br />例外が起きた場合は処理2を実行し、<br />最後に必ず処理3を実行する。 |

- 書式1

  ```ruby
  begin 
    処理1
  [rescue [エラー型] [=> e]
    処理2-1
  rescue [エラー型] [=> e]
    処理2-2
   ..]
  [ensure
    処理3]
  end
  ```

## raise, retry

### 書式

| 書式    | 概要                                              |
| ------- | ------------------------------------------------- |
| `raise` | 例外を投げる。                                    |
| `retry` | 処理を`rescue`ブロックから`begin`ブロックに移す。 |
