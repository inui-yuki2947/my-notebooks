## ruby

### コマンド

| コマンド                 | 概要                             |
| ------------------------ | -------------------------------- |
| `ruby -v, --version`     | rubyのバージョンを表示する。     |
| `ruby --help`            | rubyのヘルプを表示する。         |
| `ruby`                   | 標準入力を実行する。             |
| `ruby ファイル名 [引数]` | 指定したrubyファイルを実行する。 |
