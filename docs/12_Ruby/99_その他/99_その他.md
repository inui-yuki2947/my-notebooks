# Ruby/その他/その他

## sleep

### 書式

| 書式         | 概要                    |
| ------------ | ----------------------- |
| `sleep`      | 永久にスリープする。    |
| `sleep(sec)` | sec秒だけスリープする。 |

## 型変換

### 書式

| 書式   | 概要                 |
| ------ | -------------------- |
| `to_i` | 整数型に変換する。   |
| `to_s` | 文字列型に変換する。 |
