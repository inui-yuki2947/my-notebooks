# Ruby/その他/スクリプト

## コマンドライン引数

### 記法

| 記法      | 概要                                   |
| --------- | -------------------------------------- |
| `$0`      | コマンド名。                           |
| `ARGV[n]` | n番目のコマンドライン引数。（0始まり） |

## 標準入出力

### 記法

| 記法         | 概要                                                         |
| ------------ | ------------------------------------------------------------ |
| `gets`       | コマンドライン引数のファイル内容、または標準入力を1行読み取った結果を返す。<br />EOFに到達した際は、`nil`を返す。 |
| `STDIN.gets` | 標準入力を1行読み取った結果を返す。<br />EOFに到達した際は、`nil`を返す。 |
