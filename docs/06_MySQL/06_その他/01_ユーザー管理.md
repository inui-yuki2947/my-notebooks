# MySQL/その他/ユーザー管理

## USER, CURRENT_USER

### 記法

| 記法             | 概要                                                         |
| ---------------- | ------------------------------------------------------------ |
| `USER()`         | 接続時に指定したユーザー名とホスト名。                       |
| `CURRENT_USER()` | 実際に認証されたユーザー名とホスト名。（通常は`USER()`と同じ） |
