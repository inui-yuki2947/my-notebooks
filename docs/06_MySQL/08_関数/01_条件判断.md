# MySQL/関数/条件判断

## IS NULL

| 記法             | 概要               |
| ---------------- | ------------------ |
| `値 IS NULL`     | 値がNULLであるか。 |
| `値 IS NOT NULL` | 値がNULLでないか。 |

- `NULL`を通常の比較演算子で比較しても、**結果はNULLとなる**。

## LIKE

### 記法

| 記法                   | 概要                           |
| ---------------------- | ------------------------------ |
| `文字列 LIKE パターン` | 文字列がパターンに一致するか。 |

### パターン

| パターン | 概要                        |
| -------- | --------------------------- |
| `%`      | 任意の文字列**0文字以上**。 |
| `_`      | 任意の文字列**1文字**。     |

## REGEXP, RLIKE

### 記法

| 記法                                                         | 概要                               |
| ------------------------------------------------------------ | ---------------------------------- |
| `文字列 REGEXP 拡張正規表現`<br />`文字列 RLIKE 拡張正規表現` | 文字列が拡張正規表現に一致するか。 |
