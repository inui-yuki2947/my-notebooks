# MySQL/その他/その他

## EXPLAIN, DESCRIBE

- `EXPLAIN, DESCRIBE`文は、SQL文の実行プランを表示する。

  ```mysql
  <EXPLAIN,DESCRIBE> SQL文
  ```

## 暗号化・復号

- 暗号化には、`AES_ENCRYPT`関数を使う。

  ```sql
  AES_ENCRYPT(文字列, 暗号化鍵 [,初期化ベクトル])
  ```

- 復号には、`AES_DECRYPT`関数を使う。

  ```sql
  AES_DECRYPT(文字列, 暗号化鍵 [,初期化ベクトル])
  ```

- 暗号化鍵が暗号化時のものと異なると、`NULL`を返す。
