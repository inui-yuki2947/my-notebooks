## CONCAT()

### 記法

| 記法                  | 概要               |
| --------------------- | ------------------ |
| `CONCAT(文字列, ...)` | 文字列を結合する。 |

## LIKE, REGEXP

### 記法

| 記法                                                         | 概要                               |
| ------------------------------------------------------------ | ---------------------------------- |
| `文字列 LIKE パターン`                                       | 文字列がパターンに一致するか。     |
| `文字列 REGEXP 拡張正規表現`<br />`文字列 RLIKE 拡張正規表現` | 文字列が拡張正規表現に一致するか。 |

### パターン

| パターン | 概要                        |
| -------- | --------------------------- |
| `%`      | 任意の文字列**0文字以上**。 |
| `_`      | 任意の文字列**1文字**。     |
