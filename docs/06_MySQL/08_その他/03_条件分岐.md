# MySQL/その他/条件分岐

## IF()

### 記法

| 記法                | 概要                                                         |
| ------------------- | ------------------------------------------------------------ |
| `IF(値1, 値2, 値3)` | 値1が`TRUE`である（`0, NULL`でない）場合は値2を、そうでない場合は値3を返す。 |

## IFNULL()

| 記法               | 概要                                                      |
| ------------------ | --------------------------------------------------------- |
| `IFNULL(値1, 値2)` | 値1が`NULL`でない場合は値1を、そうでない場合は値2を返す。 |
