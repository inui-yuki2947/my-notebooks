## BEGIN, COMMIT, ROLLBACK

### 記法

| 記法                                    | 概要                                                         |
| --------------------------------------- | ------------------------------------------------------------ |
| `START TRANSACTION`<br />`BEGIN [WORK]` | 新しいトランザクションを開始する。                           |
| `COMMIT [WORK]`                         | 現在のトランザクションをコミットして、その変更を永続的なものにする。 |
| `ROLLBACK [WORK]`                       | 現在のトランザクションをロールバックして、その変更を取り消す。 |

### 例

```mysql
BEGIN;
処理1;
処理2;
...
COMMIT;
```
