# MySQL/その他/その他

## USER(), CURRENT_USER()

### 記法

| 記法             | 概要                                                         |
| ---------------- | ------------------------------------------------------------ |
| `USER()`         | 接続時に指定したユーザー名とホスト名を返す。                 |
| `CURRENT_USER()` | 実際に認証されたユーザー名とホスト名を返す。（通常は`USER()`と同じ） |

## SHOW PROCESSLIST

- `SHOW PROCESSLIST`文は、実行されているスレッドを表示する。

  ```sql
  SHOW PROCESSLIST
  ```

- PROCESS権限がない場合は、同ユーザーのスレッドのみを表示する。

## KILL

- `KILL`文は、スレッドを強制終了する。

  ```sql
  KILL プロセスリストID
  ```

## EXPLAIN, DESCRIBE

- `EXPLAIN, DESCRIBE`文は、SQL文の実行プランを表示する。

  ```mysql
  <EXPLAIN,DESCRIBE> SQL文
  ```
