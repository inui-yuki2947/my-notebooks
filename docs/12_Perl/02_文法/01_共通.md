# Perl/文法/共通

## 共通

### 文

- 文の区切りは、セミコロンが必須。

```perl
コード;
```

### コメント

```awk
# コメント
```

### シバン

```text
#!/usr/bin/perl
```

### シバン（環境依存）

```text
#!/usr/bin/env perl
```

### 文法のチェック

```perl
use strict;
use warnings;
```
