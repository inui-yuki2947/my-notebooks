# Perl/その他/標準入力処理

## 行入力演算子・ダイヤモンド演算子

### 記法

| 記法                          | 概要                                                         |
| ----------------------------- | ------------------------------------------------------------ |
| `[my $変数名 =] <ファイル名>` | 指定したファイルから、1行ずつ読み込み、変数に代入する。      |
| `[my $変数名 =] <>`           | コマンドライン引数で指定したファイルまたは標準入力から、1行ずつ読み込み、変数に代入する。 |

- `my $変数名 =`を省略すると、`$_ =`となる。
- 読み込む行がなくなると、`undef`を返す。
