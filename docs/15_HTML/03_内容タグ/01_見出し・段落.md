# HTML/内容タグ/見出し・段落

## h1, p

### タグ

| タグ                                          | 概要               |
| --------------------------------------------- | ------------------ |
| `<h1>内容</h1>`<br />...<br />`<h6>内容</h6>` | 見出し1〜6のタグ。 |
| `<p>内容</p>`                                 | 段落を表すタグ。   |

## div, span

### タグ

| タグ                | 概要                                   |
| ------------------- | -------------------------------------- |
| `<div>内容</div>`   | 特に意味を持たない**ブロック要素**。   |
| `<span>内容</span>` | 特に意味を持たない**インライン要素**。 |
