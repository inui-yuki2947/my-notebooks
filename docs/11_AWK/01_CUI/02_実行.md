# AWK/CUI/実行

## awk

### コマンド

| コマンド                                      | 概要                                                         |
| --------------------------------------------- | ------------------------------------------------------------ |
| `awk 'スクリプト' [ファイル名...]`            | 指定したファイル（省略時は標準入力）に対して、指定したスクリプトを実行する。 |
| `awk -f スクリプトファイル名 [ファイル名...]` | 指定したファイル（省略時は標準入力）に対して、スクリプトファイルの内容を実行する。 |

### オプション

| オプション      | 概要                                         |
| --------------- | -------------------------------------------- |
| `-v 変数名=値`  | 変数を定義し、スクリプト内で使用可能とする。 |
| `-F 区切り文字` | 区切り文字を指定する。                       |
