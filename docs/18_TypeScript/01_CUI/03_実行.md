# TypeScript/CUI/実行

## tsc

### コマンド

| コマンド                          | 概要                                                         |
| --------------------------------- | ------------------------------------------------------------ |
| `tsc ファイル名1 ファイル名2 ...` | 指定したファイルを全てコンパイルする。<br />（同ディレクトリに、`同ファイル名.js`ファイルを作成する） |

## tsc --init

### コマンド

| コマンド     | 概要                                  |
| ------------ | ------------------------------------- |
| `tsc --init` | `./tsconfig.json`ファイルを作成する。 |
