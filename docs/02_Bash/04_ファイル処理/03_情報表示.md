# Bash/ファイル処理/情報表示

## file

### コマンド

|コマンド|概要|
|---|---|
|`file ファイル名`|ファイルの内容をもとに、ファイルの形式を表示する。|

## wc

### コマンド

| コマンド           | 概要                                                   |
| ------------------ | ------------------------------------------------------ |
| `wc ファイル名...` | 指定したファイルの、行数・単語数・バイト数を表示する。 |

### オプション

| オプション    | 概要                   |
| ------------- | ---------------------- |
| `-l, --lines` | 行数のみ表示する。     |
| `-w, --words` | 単語数のみ表示する。   |
| `-c, --bytes` | バイト数のみ表示する。 |
