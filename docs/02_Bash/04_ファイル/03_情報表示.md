# Bash/ファイル/情報表示

## file

### コマンド

|コマンド|概要|
|---|---|
|`file ファイル名`|ファイルの内容をもとに、ファイルの形式を表示する。|

## wc

### コマンド

| コマンド           | 概要                                                   |
| ------------------ | ------------------------------------------------------ |
| `wc ファイル名...` | 指定したファイルの、行数・単語数・バイト数を表示する。 |

### オプション

| オプション    | 概要                   |
| ------------- | ---------------------- |
| `-l, --lines` | 行数のみ表示する。     |
| `-w, --words` | 単語数のみ表示する。   |
| `-c, --bytes` | バイト数のみ表示する。 |

## md5sum, shaXXsum

### コマンド

| コマンド                                           | 概要                         |
| -------------------------------------------------- | ---------------------------- |
| `md5sum [ファイル名]`<br />`shaXXsum [ファイル名]` | 入力のハッシュ値を算出する。 |

### その他

- 以下の順で、計算が遅く、安全性が高くなる。
  - md5sum
  - sha1sum
  - sha224sum
  - sha256sum
  - sha384sum
  - sha512sum
