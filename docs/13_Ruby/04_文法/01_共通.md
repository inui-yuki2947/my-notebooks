# Ruby/文法/共通

## 共通

| 項目       | 内容                  |
| ---------- | --------------------- |
| 拡張子     | `.rb`                 |
| 文の区切り | `;`（任意）           |
| インデント | スペース2個           |
| シバン     | `#!/usr/bin/env ruby` |

## コメント

### 単一行

```ruby
# コメント
```

### 複数行

```ruby
=begin
コメント
...
=end
```
