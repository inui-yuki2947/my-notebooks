# Ruby/制御構造/条件判断

## 共通

### 真偽値

| 真偽値  | 概要                                   |
| ------- | -------------------------------------- |
| `true`  | `false`と`nil`を除くオブジェクト全て。 |
| `false` | `false`と`nil`                         |

## 比較演算子

### 記法

| 記法          | 概要                                                         |
| ------------- | ------------------------------------------------------------ |
| `値1 == 値2`  | 値1と値2が**厳密な意味で**正しいか。                         |
| `値1 === 値2` | 値1と値2が**緩い意味で**正しいか。<br />（正規表現の判定や、インスタンスか否かの判定を行う） |

## その他

### メソッド

| メソッド | 概要                                                         |
| -------- | ------------------------------------------------------------ |
| `nil?`   | そのオブジェクトが`nil`であるか。                            |
| `empty?` | オブジェクトが文字列であれば、その長さが0であるか。<br />オブジェクトが配列であれば、その要素数が0であるか。 |
