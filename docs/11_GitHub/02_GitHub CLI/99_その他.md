# GitHub/GitHub CLI/その他

## gh auth

### コマンド

| コマンド        | 概要                       |
| --------------- | -------------------------- |
| `gh auth login` | GitHub CLIでログインする。 |

## gh repo

### コマンド

| コマンド       | 概要                         |
| -------------- | ---------------------------- |
| `gh repo list` | リポジトリの一覧を表示する。 |

## gh api

### コマンド

| コマンド                | 概要                    |
| ----------------------- | ----------------------- |
| `gh api エンドポイント` | GitHubのAPIを実行する。 |
