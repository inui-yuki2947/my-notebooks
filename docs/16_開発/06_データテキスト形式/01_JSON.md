## 文法基礎

### コメント

- コメントは存在しない。

### 配列

```json
[
  "aaa",
  "bbb",
  "ccc"
]
```

### ハッシュ

```json
{
  "name": "Tanaka",
  "age": 26
}
```

- キーの文字列は、**ダブルクォーテーション**で囲う。
