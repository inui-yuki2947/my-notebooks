# 開発/ビルドツール/共通

## 共通

### 概要

- ビルドツールとは、コンパイルを行ったり、jarファイルなどを生成したりするツールのこと。
- ビルドツールでは、ファイルの変更を検知し、一部分のみビルドし直す「インクリメンタルビルド」をサポートする。

- 例

| 種類   | ビルドファイル | 言語   |
| ------ | -------------- | ------ |
| Make   |                |        |
| Ant    | `build.xml`    |        |
| Gradle | `pom.xml`      | Groovy |
| Maven  | `build.gradle` | XML    |
