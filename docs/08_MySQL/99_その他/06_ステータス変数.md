# MySQL/その他/ステータス変数

## ステータス変数

### 概要

- ステータス変数は、サーバーの操作についての情報を提供する変数。

### 書式

| 書式          | 概要                             |
| ------------- | -------------------------------- |
| `SHOW STATUS` | ステータス変数の一覧を表示する。 |
