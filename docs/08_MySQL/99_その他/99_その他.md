# MySQL/その他/その他

## SHOW PROCESSLIST

- `SHOW PROCESSLIST`文は、実行されているスレッドを表示する。

  ```sql
  SHOW PROCESSLIST
  ```

- PROCESS権限がない場合は、同ユーザーのスレッドのみを表示する。

## KILL

- `KILL`文は、スレッドを強制終了する。

  ```sql
  KILL プロセスリストID
  ```

## EXPLAIN, DESCRIBE

- `EXPLAIN, DESCRIBE`文は、SQL文の実行プランを表示する。

  ```mysql
  <EXPLAIN,DESCRIBE> SQL文
  ```
