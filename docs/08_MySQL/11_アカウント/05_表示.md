# MySQL/アカウント/表示

## mysql.user

### テーブル

| テーブル     | 概要                                 |
| ------------ | ------------------------------------ |
| `mysql.user` | アカウント情報が格納されたテーブル。 |

### カラム

| カラム | 概要         |
| ------ | ------------ |
| `user` | ユーザー名。 |
| `host` | ホスト名。   |

### 例

```mysql
SELECT user, host FROM mysql.user;
```

## USER(), CURRENT_USER()

### 記法

| 記法             | 概要                                                         |
| ---------------- | ------------------------------------------------------------ |
| `USER()`         | 接続時に指定したユーザー名とホスト名を返す。                 |
| `CURRENT_USER()` | 実際に認証されたユーザー名とホスト名を返す。（通常は`USER()`と同じ） |
