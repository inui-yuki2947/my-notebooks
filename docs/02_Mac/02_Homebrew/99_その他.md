# Mac/Homebrew/その他

## brew services

### コマンド

| コマンド                                  | 概要                                                   |
| ----------------------------------------- | ------------------------------------------------------ |
| `brew services`<br />`brew services list` | サービスの一覧を表示する。                             |
| `brew services start サービス`            | サービスを開始し、ログイン時にも起動させるようにする。 |
| `brew services stop サービス`             | サービスを終了し、ログイン時にも起動しないようにする。 |
| `brew services restart サービス`          | サービスを再起動する。                                 |
