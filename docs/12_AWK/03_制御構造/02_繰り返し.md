# AWK/制御構造/繰り返し

## while

### 記法

| 記法                     | 概要                                                         |
| ------------------------ | ------------------------------------------------------------ |
| `while (条件) {処理}`    | 条件を満たす場合に、繰り返し処理を実行する。（先に条件評価） |
| `do {処理} while (条件)` | 条件を満たす場合に、繰り返し処理を実行する。（後に条件評価） |

## for

### 記法

| 記法                                  | 概要                                                         |
| ------------------------------------- | ------------------------------------------------------------ |
| `for (初期化式; 条件; 増減式) {処理}` | 初期化式を実行した後、条件を満たす限り、処理及び増減式を実行する。 |
| `for (変数名 in 配列名) {処理}`       | 配列の要素について、繰り返し処理を実行する。                 |
