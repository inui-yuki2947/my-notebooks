# バックエンド開発/その他/バックエンド全般

## 共通

- メソッド

| メソッド | 概要                                                         |
| -------- | ------------------------------------------------------------ |
| GET      | リソースを取得する。                                         |
| POST     | リソースを作成する。                                         |
| PUT      | リソースを上書きする。（既存リソースが存在しなければ作成する） |
| PATCH    | リソースを一部更新する。                                     |
| DELETE   | リソースを削除する。                                         |
